# 🍔 Yummy Express – Online Food Ordering System

A full-stack food delivery web application built using the MERN stack.  
Users can browse food items, add to cart, and place orders. Admin can manage menu and orders.

---

## 🚀 Features

- Browse food items by category
- Add / remove items from cart
- Place orders
- User authentication (login/signup)
- Admin panel for managing food & orders

---

## 🛠️ Tech Stack

- Frontend: React.js, Vite, CSS
- Backend: Node.js, Express.js
- Database: MongoDB
