# 🍔 Yummy Express – Online Food Ordering System

A full-stack food delivery web application built using the MERN stack. Users can browse food items, add to cart, place orders, and manage their account. Includes an admin panel for managing menu and orders.

---

## 🚀 Features

### 👤 User Side
- Browse food items by category
- Add/remove items from cart
- Place orders
- User authentication (login/signup)

### 🛠️ Admin Panel
- Add / update / delete food items
- Manage orders
- View user activity

---

## 🧑‍💻 Tech Stack

- **Frontend:** React.js, Vite, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Others:** REST APIs, JWT Authentication
